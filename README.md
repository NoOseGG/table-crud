# React Table CRUD Test

Простой проект на **React + TypeScript + Ant Design + React Query**, демонстрирующий работу с таблицей пользователей и CRUD операциями через mock API.

## Функционал

- Отображение списка пользователей в таблице  
- Добавление нового пользователя (новые элементы добавляются в начало таблицы)  
- Редактирование существующего пользователя через модальное окно  
- Удаление пользователя  
- Валидация форм (Ant Design Form + DatePicker + InputNumber)  
- Автоматическое обновление данных через React Query

## Технологии

- React + TypeScript  
- Ant Design  
- React Query  
- Axios  
- Mock API (mockapi.io)  
- Day.js (для работы с датами)
